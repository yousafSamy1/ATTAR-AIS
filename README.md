# 🌿 Attar AIS – نظام إدارة محل العطار

تطبيق **سطح مكتب** متكامل لإدارة محل العطار، مبني بـ **Python** و **PyQt6** و **SQLite**.

## 📂 المميزات
- إدارة المنتجات والمخزون (توابل، أعشاب، مستلزمات)
- تسجيل وتتبع **المبيعات والمشتريات**
- إدارة **العملاء والموردين**
- تتبع **المصروفات**
- **القيود المحاسبية** التلقائية
- **تقارير مالية** شاملة (أرباح وخسائر، كشف حسابات)
- **داش بورد** تفاعلي بتصميم بني ذهبي يعبر عن هوية العطارة

## 🛠 التقنيات المستخدمة
- **Python 3.13**
- **PyQt6** — واجهة المستخدم
- **SQLite** — قاعدة البيانات

## 🚀 تشغيل التطبيق
```
python main.py
```
أو تشغيل ملف `dist/AttarAIS.exe` مباشرةً.
